Breaking Ball

Exciting Brick-Breaking Game Developed in Java

About the Game

Welcome to the Breaking Ball game! This game is a fun and exciting brick-breaking game where you control a paddle to bounce a ball and break bricks. Try to break all the bricks to win the game!

For more details and to view the source code, visit:

GitHub repository

Features

  • Smooth and Responsive Controls: Enjoy seamless and intuitive interactions.
  • Increasing Difficulty Levels: Experience a gradual increase in challenge as you progress.
  • Attractive Graphics: Visually appealing design enhances the gaming experience.
  • Score Display: Scores displayed at the end of the quiz session.

Instructions

To play the game, download the JAR file and run it on your computer.
Use the arrow keys to move the paddle left and right. Your goal is to prevent the ball from falling off the screen and to break all the bricks.

How to Run the Game

1. Clone the Repository

git clone git@github.com:Kvr-10/Breaking-Ball-Game.git

2. Navigate to the Artifacts Directory

After cloning the repository, navigate to the following path:

Breaking-Ball-Game/out/artifacts/Breaking_ball_jar/Breaking_ball.jar

3. Run the Game

Locate the Breaking_ball.jar file in the directory.

Double-click on Breaking_ball.jar to start the game.
